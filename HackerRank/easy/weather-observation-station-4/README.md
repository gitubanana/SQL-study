### ABS
숫자의 절댓값을 반환하는 함수<br>
- **기본 문법**<br>
  ```sql
  ABS(number)
  -- number : 숫자(필수)
  ```
- **예시**<br>
  ```sql
  SELECT ABS(-243.5);
  -- 243.5
  ```
- **동작 환경**<br>
  MySQL 4.0부터<br>
