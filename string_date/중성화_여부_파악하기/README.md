### 문제 간단 설명
ANIMAL_INS 테이블에서<br>
ANIMAL_ID와 NAME, 중성화 여부를 조회하는 SQL문 작성<br>
중성화 여부는<br>
    SEX_UPON_INTAKE에 'Neutered' 또는 'Spayed'라는 단어가 들어있으면 'O'<br>
    아니면 'X'라고 표시.<br>
결과는 ANIMAL_ID 순으로 정렬.<br>
