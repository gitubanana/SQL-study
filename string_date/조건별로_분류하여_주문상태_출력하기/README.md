### 문제 간단 설명
FOOD_ORDER 테이블에서<br>
ORDER_ID, PRODUCT_ID, OUT_DATE, 출고여부를 조회하는 SQL문을 작성.<br>
출고여부는 OUT_DATE가
    5월 1일까지이면 출고완료<br>
    이 후 날짜는 출고대기<br>
    미정이면 출고미정으로 출력.<br>
결과는 ORDER_ID를 기준으로 오름차순 정렬.<br>
