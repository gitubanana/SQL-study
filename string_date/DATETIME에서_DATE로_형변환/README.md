### 문제 간단 설명
ANIMAL_INS 테이블에서<br>
모든 레코드에 대해, ANIMAL_ID, NAME과 들어온 날짜를 조회하는 SQL문을 작성.<br>
들어온 날짜는 날짜(년-월-일)만 보여주고, 칼럼명은 `날짜` 로.<br>
이때 결과는 아이디 순으로 조회.<br>
