### 문제 간단 설명
ANIMAL_INS 테이블에서<br>
NAME이 두 번 이상 쓰인 NAME과 해당 이름이 쓰인 횟수를 조회하는 SQL문을 작성.<br>
이때 결과는 NAME이 없는 동물은 집계에서 제외하며, 결과는 NAME 순으로 조회.<br>

### 새로 배운 점
GROUP BY한 칼럼의 값이 NULL이어도 그룹으로 묶인다.<br>
