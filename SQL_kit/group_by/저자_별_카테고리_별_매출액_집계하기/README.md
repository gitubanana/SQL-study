### 헷갈린 점
TOTAL_SALES를 구할 때, `SUM(PRICE * SALES)`로 해야 하는데, `PRICE * SUM(SALES)`로 해서 많이 헤맸다.<br>
둘이 얼핏 보면 같다고 생각할 수 있는데, TOTAL_SALES가 저자별, 카테고리별 매출액이어서 책의 PRICE가 달라질 수 있어서 SUM안에 집어넣어야 한다...<br>
