### 문제 간단 설명
CAR_RENTAL_COMPANY_CAR 테이블에서<br>
'통풍시트', '열선시트', '가죽시트' 중 하나 이상의 옵션이 포함된 자동차가 CAR_TYPE별로 몇 대인지 출력하는 SQL문 작성.<br>
