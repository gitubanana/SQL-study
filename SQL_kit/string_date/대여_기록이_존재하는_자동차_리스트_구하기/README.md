### 느낀 점
나는 GROUP BY로 풀었지만, DISTINCT를 이용한 풀이를 생각하지 못했다.<br>
GROUP BY를 이용해도 중복을 제거할 수 있어서 DISTINCT를 자꾸 까먹는 것 같다.<br>
